美化

默认使用Unity桌面

  • unity-tweak-tool
    安装sudo apt-get install unity-tweak-tool
  • 也可以通过安装ubuntu-tweak来实现主题更换
sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak
  • 主题网址http://gnome-look.org/ 将解压的主题文件夹拷贝到/usr/share/themes/目录下,然后使用unity-tweak-tool选择;
  • icon网址https://github.com/NitruxSA/ardis-icon-theme 将解压的主题文件夹拷贝到/usr/share/icons/目录下,然后使用unity-tweak-tool选择;或者直接命令安装再设置;
